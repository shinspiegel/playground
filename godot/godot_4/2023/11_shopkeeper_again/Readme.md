- **Assets**
	- Hard assets and materials. As it from the origin.
- **Entities*
	- Objects to use in the levels
- **Levels**
	- Levels for the player to wanter around
- **Managers**
	- Singletons
- **Shaders**
	- All the shader material used
- **Main.[tscn/gd]**
	- Entry point for the game
